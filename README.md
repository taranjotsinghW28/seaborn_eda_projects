📊 EDA — Titanic & Tips Datasets
Exploratory Data Analysis and Machine Learning on two classic Seaborn datasets — Titanic and Tips.

📁 Datasets
Dataset
titanic -Passenger survival data from the Titanic disaster
tips-Restaurant tipping behavior data

🔄 Pipeline
Data Loading → Null Values → Duplicates → Outliers → Scaling → Visualization → ML Model

📌 Steps Covered

Data Loading — Loaded using seaborn.load_dataset()
Null Value Analysis — Identified and handled missing values
Duplicate Detection — Checked and removed duplicates
Outlier Detection — Used IQR method with box plots
Feature Scaling — Applied StandardScaler / MinMaxScaler
Visualization — Plots, heatmaps, correlation matrices
ML Model — Built and evaluated classification/regression models


🛠️ Tech Stack
Python Pandas NumPy Seaborn Matplotlib Scikit-learn
